À Propos du Projet

J'ai développé ce projet pour offrir une solution de gestion de mots de passe sécurisée et entièrement hors ligne. Cette application peut être installée sur n'importe quel ordinateur Windows, macOS ou Linux.

Son rôle ? Générer des mots de passe ultra-sécurisés et les stocker dans un coffre-fort chiffré avec un algorithme de chiffrement militaire (AES-128).

L'utilisateur peut générer des mots de passe personnalisables ou des phrases de passe mémorisables. L'application intègre également une fonctionnalité de vérification de sécurité qui interroge l'API "Have I Been Pwned" pour détecter si un mot de passe a été compromis dans une fuite de données.

Fonctionnalités Clés

Générateur de mots de passe

Critères personnalisables (longueur, caractères spéciaux, etc.)

Coffre-fort chiffré

Chiffrement AES-128 et dérivation de clé Argon2id

Vérification de sécurité

API Have I Been Pwned pour détecter les fuites

Verrouillage automatique

Sécurité renforcée après inactivité

Import/Export

Données en CSV et JSON pour sauvegardes

Historique

Suivi des mots de passe pour chaque compte

Technologies Utilisées

Python CustomTkinter Cryptography Argon2 SQLite
