# Dossier Downloads

Ce dossier contient les fichiers ZIP téléchargeables pour les projets du portfolio.

## Structure des fichiers :

Tous les projets sont distribués sous forme de fichiers ZIP contenant l'application et toutes les ressources nécessaires.

## Instructions pour ajouter vos projets :

### Pour ajouter un nouveau projet :

1. **Créez un dossier** contenant votre application (.exe) et toutes les ressources nécessaires (assets, dossiers, etc.)
2. **Compressez le dossier en ZIP** (clic droit > Envoyer vers > Dossier compressé sur Windows)
3. **Renommez le fichier ZIP** selon la convention de nommage :
   - `ZephyrPasswordManager.zip` - Gestionnaire de mots de passe ✅ (AJOUTÉ)
   - `SiteRestaurant.zip` - Site web restaurant (À ajouter)
   - `ApplicationMobileMachina.zip` - Application mobile IA (À ajouter)
4. **Placez le fichier ZIP** dans ce dossier `downloads/`

## Structure actuelle :

```
downloads/
├── README.md (ce fichier)
├── ZephyrPasswordManager.zip ✅ (PRÊT - contient .exe + ressources)
├── SiteRestaurant.zip (À AJOUTER quand le projet sera terminé)
├── ApplicationMobileMachina.zip (À AJOUTER quand le projet sera terminé)
├── placeholder-machina.txt (à supprimer une fois le ZIP ajouté)
└── placeholder-restaurant.txt (à supprimer une fois le ZIP ajouté)
```

## Notes importantes :

- Les fichiers ZIP seront téléchargés directement par les visiteurs du site
- Les utilisateurs devront extraire le ZIP sur leur ordinateur
- Assurez-vous que les noms de fichiers correspondent exactement aux noms ci-dessus
- Les boutons de téléchargement sont déjà configurés pour pointer vers ces fichiers ZIP
- Pour les projets non terminés, un message s'affiche indiquant que le téléchargement n'est pas encore disponible

## Comment créer un fichier ZIP sur Windows :

1. Rassemblez tous les fichiers nécessaires dans un dossier
2. Clic droit sur le dossier
3. Sélectionnez "Envoyer vers" > "Dossier compressé"
4. Renommez le fichier .zip selon la convention ci-dessus
5. Copiez-le dans ce dossier `downloads/`
