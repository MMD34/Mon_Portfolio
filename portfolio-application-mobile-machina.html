Application mobile

Cette application mobile innovante exploite la puissance de l'intelligence artificielle pour analyser des visages et estimer l'âge des personnes à partir d'une simple photographie. Développée comme projet de recherche et d'apprentissage, elle démontre l'intégration réussie d'algorithmes de vision par ordinateur dans une application mobile native multiplateforme.

Fonctionnalités principales :
• Capture photo en temps réel : L'utilisateur peut prendre une photo directement depuis l'application ou charger une image existante de sa galerie
• Analyse par IA : Un algorithme de deep learning analyse les caractéristiques faciales pour estimer l'âge avec précision
• Résultats instantanés : L'estimation s'affiche en quelques secondes avec une interface intuitive
• Interface moderne : Design épuré et expérience utilisateur optimisée pour iOS et Android

Architecture et technologies :
L'application est développée en C# avec Xamarin.Forms, permettant un déploiement multiplateforme efficace sur iOS et Android à partir d'une base de code unique. Le modèle d'intelligence artificielle utilise des réseaux de neurones convolutionnels (CNN) pré-entraînés pour la détection et l'analyse faciale.

L'architecture suit le pattern MVVM (Model-View-ViewModel) garantissant une séparation claire des responsabilités et facilitant la maintenance du code. L'accès à la caméra et à la galerie utilise les API natives des plateformes via des services de dépendance.

Défis techniques relevés :
• Optimisation des performances pour l'analyse en temps réel sur mobile
• Gestion des permissions caméra/galerie sur différentes versions d'OS
• Intégration du modèle d'IA avec gestion de la mémoire optimisée
• Interface responsive s'adaptant à toutes les tailles d'écran

Résultat :
Une application mobile complète et fonctionnelle démontrant ma maîtrise du développement mobile multiplateforme et de l'intégration d'algorithmes d'intelligence artificielle dans des applications pratiques.

Technologies utilisées: C#, Xamarin.Forms, .NET, Machine Learning, Vision par ordinateur, MVVM Pattern
